An asynchronous, Anonymous, and End to End Encrypted chat app focused on security and privacy. This application used the Tornado Framework, Python, and WebSockets to power the backend and ReactJS for the frontend. BrowserOS
Okay, I get it. It's not an operating system. It doesn't even remotely try to be an operating system. But the name sticks. This is a web app written with ReactJS that tries to simulate a computer desktop with different windows and applications. It features a windowing system, a persistent filesystem, an application launcher, and a fake terminal with limited functionality
